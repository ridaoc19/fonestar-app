<h2>Descripción del Proyecto</h2>
<p>
	El Gestor de Modelos de Traducción Fonestar es una aplicación web diseñada para gestionar y editar
	prompts de traducción para diferentes modelos de lenguaje. La aplicación permite a los usuarios
	ver, actualizar y gestionar prompts de traducción a través de una interfaz fácil de usar.
</p>
<p>Las características principales de la aplicación incluyen:</p>
<ul>
	<li>Obtención y visualización de prompts de traducción desde una API REST.</li>
	<li>Edición de respuestas de traducción en múltiples idiomas (inglés, francés, portugués).</li>
	<li>Actualización de feedback para cada prompt.</li>
	<li>Autenticación mediante un mecanismo seguro de token API.</li>
</ul>
